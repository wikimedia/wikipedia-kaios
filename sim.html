About the simulator

  • This is a BETA version of the simulator for the Wikipedia app on KaiOS.
  • To learn more about the app, visit the project page
  • If you come across issues with the simulator, please let us know on the talk page or file an issue in Phabricator

How to use the phone

  • This is a non-touch phone. Use the keypad to navigate the app
  • Use the Enter button to go forward (twice to start).
  • Use the up/down keys to page through an article
  • Use the left/right keys to cycle through the article links
  • Use the red key to go back
